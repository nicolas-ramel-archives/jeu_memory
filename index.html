Jeu de mémoire

La règle est simple, vous devez associer les fruits ensemblent. Pour cela vous cliquez sur une zone grise pour révéler un fruit. Puis vous cliquez sur une deuxième zone grise. Si les fruits sont identiques, félicitation vous avez fait une bonne association. Si ce n'est pas le cas, garder dans votre mémoire la position des fruits car il vous faudra en rechercher d'autres.
Vous avez 1 minute pour associer tous les fruits entre eux.
Dès que vous êtes prêt, cliquez sur "Commencer le jeu" avec le niveau de jeu que vous souhaitez.
